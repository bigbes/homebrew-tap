# Bigbes Tap

## How do I install these formulae?

`brew install bigbes/tap/<formula>`

Or `brew tap bigbes/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "bigbes/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
